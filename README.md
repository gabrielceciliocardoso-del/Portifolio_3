# Controle de Estoque Ionic/Angular

Sistema de controle de estoque criado com Ionic e Angular.

## Como rodar

```bash
npm install
npm start
```

Depois abra o endereco exibido pelo Angular, normalmente `http://localhost:4200`.

## Acesso inicial

- Usuario: `admin`
- Senha: `admin123`

## Recursos

- Login com usuario e senha.
- Cadastro, edicao e exclusao de usuarios.
- Cadastro, edicao, exclusao e busca de produtos.
- Cadastro, edicao e exclusao de clientes.
- Registro de vendas com cliente, produtos, quantidades, forma de pagamento, data e horario.
- Baixa automatica do estoque ao finalizar uma venda.
- Caixa com entradas automaticas das vendas e entradas manuais.
- Relatorios de vendas do mes, valor vendido, saldo em caixa e quantidade em estoque.

Os dados ficam salvos no `localStorage` do navegador.
